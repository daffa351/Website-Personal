01 — Tentang

Berpikir dalam sistem

Saya sedang menempuh pendidikan di jurusan Teknik Mekatronika, bidang yang mempertemukan mekanik, elektronika, dan pemrograman menjadi satu kesatuan. Bagi saya, mekatronika adalah cara berpikir: bagaimana sebuah sistem mengindra kondisi di sekitarnya, mengambil keputusan, lalu bergerak dengan tepat.

Saya memiliki ketertarikan mendalam pada dunia manufaktur dan otomasi. Saya tertantang oleh proses troubleshooting—mulai dari mengatasi pembacaan sensor yang eror hingga memastikan otomatisasi berjalan sempurna.

03 — Proyek

Sistem yang sudah dibangun

P.01

RAK TAS DENGAN TEKNIK LAS

Proyek ini merancang dan mempabrikasi rak tas berbahan besi yang kokoh untuk mengatasi masalah penyimpanan barang yang tidak tertata. Menggunakan mesin las FCAW, proyek ini mengintegrasikan sistem mekanik struktural yang mampu menahan beban berat secara merata. Tantangan utama berupa deformasi panas pada besi tipis berhasil diatasi dengan teknik penyesuaian arus (amperage) dan metode pengelasan bertahap. Hasil akhirnya adalah rak tas ergonomis yang mampu menampung beban dengan hasil sambungan las yang rapi, kuat, dan minim cacat.

  • Mesin Las FCAW
  • Mesin Las SMAW
  • Sistem Mekanik
P.02

SISTEM KONTROL WITH APK

Proyek ini membangun sistem otomasi dan monitoring lingkungan jarak jauh berbasis IoT. Sistem bekerja dengan membaca data suhu dan kelembapan via sensor DHT11 pada Arduino, lalu diproses oleh Python untuk analisis data, dan dikontrol secara real-time melalui aplikasi Blynk. Tantangan teknis terbesar berupa latensi data dan sinkronisasi antar-platform berhasil diselesaikan melalui optimasi komunikasi serial serta penggunaan Virtual Pins pada Blynk. Proyek ini menghasilkan sistem kontrol dengan waktu respons LED di bawah 1 detik untuk mengetahui jika suhu tinggi maka LED akan menyala dan jika suhu rendah maka LED akan mati.

  • Arduino
  • Python
  • Sensor DHT11
  • Led
  • Apk Blynk
P.03

WEB PORTOFOLIO

Proyek ini saya buat untuk mendigitalisasi data pribadi, riwayat pendidikan, dan hasil praktik saya ke dalam satu wadah online yang rapi. Prosesnya dimulai dari merancang tata letak, masuk ke tahap coding menggunakan Visual Studio Code, melakukan debugging tampilan lewat Inspect Element, hingga mengelola source code menggunakan Git dan GitHub sampai siap di-hosting. Di proyek ini, saya menggunakan HTML untuk struktur web, CSS untuk desain responsifnya, dan JavaScript untuk menambahkan fitur interaktif.Tantangan terbesar di proyek ini justru ada pada pengembangan soft skill. Saya belajar banyak tentang komunikasi visual—bagaimana menyusun informasi diri yang padat menjadi tampilan yang simpel dan mudah dipahami orang lain. Selain itu, problem solving saya benar-benar diasah secara mandiri saat harus memecahkan error atau bug kode, ditambah lagi manajemen waktu yang ketat agar proyek ini selesai sesuai target.

  • HTML
  • JavaScript
  • CSS
  • Hosting
P.04

SISTEM KONTROL BERBASIS INDUSTRI

Proyek ini saya buat untuk mensimulasikan sistem conveyor pemotong kayu otomatis yang aman, presisi, dan efisien sesuai standar pabrik. Proses pengerjaannya dimulai dari merancang logika program diagram tangga (ladder diagram) pada CX-Programmer untuk mengatur kecepatan conveyor dan interlok pisau pemotong, lalu dilanjutkan dengan membuat desain visualisasi HMI pada CX-Designer agar operator bisa memonitor jalannya mesin. Seluruh sistem kontrol ini diintegrasikan ke perangkat PLC sebagai otak utama penggerak motor penggerak dan silinder pemotong. Tantangan utama di proyek ini adalah menyinkronkan waktu antara sensor pembaca panjang kayu dengan pergerakan pisau pemotong agar hasil potongannya pas. Selain itu, saya juga harus melatih ketelitian saat proses debugging karena sempat ada salah alamat I/O yang membuat program tidak jalan. Lewat proyek ini, saya jadi lebih paham cara kerja sistem otomatisasi industri, sekaligus melatih kemampuan troubleshooting dan kesabaran dalam menyelesaikan masalah kode program.
